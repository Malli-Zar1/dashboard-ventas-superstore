# 📊 Dashboard Interactivo de Ventas — Superstore

## Descripción
Dashboard de análisis de ventas retail usando Python y Plotly.
Visualizaciones interactivas que permiten explorar patrones de ventas,
rentabilidad y comportamiento de clientes.

## 🛠️ Herramientas utilizadas
- Python (Pandas)
- Plotly Express
- Google Colab

## 📊 Visualizaciones incluidas
- **Ventas por Sub-Categoría** — Gráfica de barras horizontal interactiva
- **Tendencia de Ventas por Año** — Líneas por categoría 2014-2017
- **Ventas vs Ganancia por Estado** — Scatter plot con tamaño y color
- **Top 10 Clientes más Rentables** — Ranking de clientes por ganancia

## 📈 Principales hallazgos
- **Phones y Chairs** son las sub-categorías con mayores ventas
- **Technology** es la categoría de mayor crecimiento 2014-2017
- Algunos estados tienen ventas altas pero ganancia negativa
- El top 10 de clientes representa una parte clave de la rentabilidad
